# 0008: ダッシュボード・概要画面

## 概要

コンテナ/イメージ/ボリューム/ネットワークの状況を横断的に俯瞰できる概要画面（ダッシュボード）
と、稼働中コンテナのCPU使用率・メモリ使用量の可視化を提供する。
Docker Desktopの「Dashboard」トップ画面に相当し、「Stats」表示についてはCPU・メモリの
基本メトリクスに絞ったサブセットを提供する（Network I/O、Block I/O、PIDs数等の
詳細メトリクスは含まない）。

## ユーザー価値

- アプリ起動直後に、現在の全体状況（稼働中コンテナ数、イメージ数、ボリューム数等）を
  一目で把握できる。
- どのコンテナがCPU・メモリを多く使っているかを確認し、問題のあるコンテナに
  気づきやすくなる。

## スコープ

- 概要画面として、以下のサマリ情報を表示する。件数は、対応する各一覧画面
  （[0001](0001-container-list-basic-operations.md)、[0003](0003-image-management.md)、
  [0005](0005-volume-management.md)、[0006](0006-network-management.md)）で
  ユーザーに見える対象と一致させる（例: ネットワーク数はシステムネットワークを
  含む、0006の一覧表示対象と同じ数）。
  - 稼働中/停止中コンテナ数
  - ローカルイメージ数
  - ボリューム数
  - ネットワーク数
- 稼働中コンテナごとのCPU使用率・メモリ使用量を表示する。稼働中コンテナが
  存在しない場合は、その旨が分かる表示にする。
- コンテナ/イメージ/ボリューム/ネットワークが起動・停止・作成・削除される等、
  対象リソースの状態が変化した場合、概要画面を表示中であれば、サマリ件数・
  CPU/メモリ表示にその変化が反映される（自動反映、または画面上での明示的な
  再読込操作のいずれかにより、常に手動でアプリを再起動しなくても最新状態を
  確認できる）。
- 概要画面のコンテナ一覧（Stats表示対象）から、該当コンテナの詳細画面
  （[0004](0004-container-detail-exec.md)）またはログ画面
  （[0002](0002-container-log-viewer.md)）へ遷移できる。
- 概要画面から各一覧画面（コンテナ一覧、イメージ一覧、ボリューム一覧、
  ネットワーク一覧）へ遷移できる。

## 非スコープ

- Network I/O、Block I/O、PIDs数等、CPU・メモリ以外のリソースメトリクス
  （将来検討）。
- リソース使用状況の履歴・グラフ推移の保存（現在値の表示に留め、時系列保存・
  グラフ化は将来検討）。
- アラート・しきい値通知機能（CPU使用率が一定を超えたら通知する等）。
- ホストマシン全体（WSL全体）のリソース使用状況表示。本仕様はコンテナ単位に限定する。
- 個別コンテナのリソース使用状況に関する詳細な調査・分析UI。問題のありそうな
  コンテナを見つけた後の詳細調査は、コンテナ詳細（0004）・ログ（0002）に委譲する。

## 受け入れ基準

- [ ] アプリ起動後、概要画面に遷移すると、稼働中/停止中コンテナ数、イメージ数、
      ボリューム数、ネットワーク数が表示され、それぞれ対応する一覧画面での
      表示対象数と一致する。
- [ ] 稼働中のコンテナについて、CPU使用率・メモリ使用量がコンテナごとに表示される。
- [ ] 稼働中のコンテナが1件も存在しない場合、Stats表示領域がその旨（対象なし）
      分かる表示になる。
- [ ] コンテナを起動/停止する、またはイメージ/ボリューム/ネットワークを作成/削除
      すると、概要画面のサマリ件数・Stats表示に変化が反映される（自動反映、
      または明示的な再読込操作のいずれか）。
- [ ] 概要画面のStats表示から、対象コンテナの詳細画面またはログ画面へ遷移できる。
- [ ] 概要画面から、コンテナ一覧・イメージ一覧・ボリューム一覧・ネットワーク一覧の
      各画面へ遷移できる。
- [ ] 対象となるコンテナ/イメージ/ボリューム/ネットワークが1つも存在しない場合、
      サマリが0件であることが分かる表示になる。
- [ ] リソース使用状況の取得に失敗した場合、エラーの理由がユーザーに伝わる。
- [ ] サマリ件数の一部（例: イメージ数のみ）の取得に失敗した場合でも、取得できた
      他の項目は表示され、失敗した項目については失敗したことが分かる。
